# Korean Learning Roadmap – Conversation & Technology Focus

This roadmap emphasizes **basic conversational ability** while also allocating time to **technology‑related Korean content** (tech podcasts, articles, and terminology). The schedule is flexible; you can adapt the length of each phase to your own pace.

---

## Phase 1 – Foundations (Weeks 1‑2)

| Goal | Resources | Daily Routine |
|------|-----------|----------------|
| **Hangul mastery** – be able to read/write all letters | • YouTube: *Learn Hangul in 5 minutes* <br>• Anki deck: *Hangul Basics* | 5 min Anki review → write 5 random words by hand
| **Core pronunciation** – basic vowels & consonants | • TTMIK Lesson 1 (audio) <br>• Forvo (listen to native speakers) | 5 min listening → repeat aloud
| **Simple greetings & intro** – “안녕하세요”, “제 이름은 … 입니다” | • Talk To Me In Korean (TTMIK) greetings lesson <br>• Language exchange app – send a hello message | Write a short self‑intro, record yourself, compare to native audio |
| **Tech‑related vocab starter** – words like *컴퓨터*, *인터넷*, *앱*, *코드* | • Memrise “Korean Tech Basics” deck (free) | Review 5 tech words each day |

**Progress check** – After two weeks you should read/write Hangul fluently and introduce yourself.

---

## Phase 2 – Everyday Conversation (Weeks 3‑6)

| Goal | Resources | Daily Routine |
|------|-----------|----------------|
| **Daily phrases** – ordering food, asking directions, small talk | • TTMIK Lessons 2‑4 <br>• “Korean Phrasebook for Travelers” PDF (resources/textbooks) | Choose 3 phrases, practice with a language partner (HelloTalk) |
| **Listening comprehension** – short Korean clips (2‑3 min) | • KoreanClass101 beginner podcasts <br>• Naver news headlines (audio version) | Listen, then write a 1‑sentence summary in Korean |
| **Tech news digest** – read simple tech articles in Korean | • *TechCrunch Korea* (Korean version) <br>• *디지털데일리* (tech news) – pick one short article per day | Highlight unknown words, add them to Anki (tech tag) |
| **Speaking practice** – 5‑minute voice note on a daily topic | • Record on HelloTalk or Tandem, request correction | Review feedback, repeat the sentence |

**Progress check** – Ability to hold a 2‑minute conversation about daily life and describe basic tech concepts.

---

## Phase 3 – Intermediate Conversational Fluency (Weeks 7‑12)

| Goal | Resources | Daily Routine |
|------|-----------|----------------|
| **Extended dialogues** – discuss hobbies, work, news | • TTMIK Lessons 5‑7 (role‑play) <br>• “Korean Conversation” workbook (resources/textbooks) | Role‑play with a partner; record and get feedback |
| **Tech‑focused listening** – podcasts, YouTube tech channels (Korean) | • *여기서 프로그래밍* (Korean programming podcast) <br>• *코딩셰프* YouTube channel | 10‑minute listening, take 3 key‑point notes in Korean |
| **Reading** – simple tech documentation (e.g., GitHub README in Korean) | • Korean‑translated Selenium docs <br>• Open‑source project READMEs on GitHub (search `language:ko`) | Highlight terms, add to Anki, try to explain them aloud |
| **Writing** – short blog post or journal entry about a tech topic (150‑200 words) | • Use Lang‑8 for corrections | Write once per week, revise with feedback |

**Progress check** – Comfortable discussing tech topics (e.g., “제가 파이썬으로 웹 스크래핑을 배우고 있어요”) and can understand beginner‑level Korean tech videos.

---

## Phase 4 – Advanced Conversation & Specialty (Months 4‑6)

| Goal | Resources | Weekly Routine |
|------|-----------|----------------|
| **Specialized tech vocab** – cloud, AI, data science terms | • Custom Anki deck (create from Korean tech blogs) <br>• *Korean Wikipedia* pages on AI, Cloud, etc. | Add 20 new terms per week, review regularly |
| **In‑depth listening** – 20‑30 min tech talks, webinars | • *AI Korea* webinars (Korean) <br>• *Google Cloud Korea* talks (YouTube) | Summarize talk in Korean, discuss with a partner |
| **Debate & opinion** – express viewpoints on tech trends | • Prepare a 2‑minute argument in Korean each week | Record, get feedback, refine |
| **Cultural immersion** – watch Korean dramas **with Korean subtitles** and note idioms | • Any popular K‑drama on Netflix (Korean subtitles) | Note 5 new idioms per episode, add to Anki |

**Final goal** – Hold a 5‑minute fluent conversation on a technical subject, understand native‑speed tech podcasts, and write coherent Korean tech posts.

---

## Supporting Tools & Habits

- **Anki** – main vocab system (tags: `hangul`, `basic`, `tech`).
- **HelloTalk / Tandem** – language exchange; set partner preference to “tech interested”.
- **Lang‑8** – get writing corrections.
- **Daily journal** – 2‑sentence entry in Korean (focus on today’s tech thing you learned).
- **Consistency** – Even 15 min a day beats occasional long sessions.
- **Feedback loop** – Actively request corrections; incorporate them immediately.

---

### Quick‑Start Checklist
1. Install **Anki** and create three decks (Hangul, Basic, Tech).  
2. Sign up on **HelloTalk** and set “learning Korean” + “technology” interests.  
3. Add the first **TTMIK** lesson to your playlist and watch the video.  
4. Subscribe to a Korean tech podcast (e.g., *여기서 프로그래밍*).  
5. Write today’s first journal entry (even a single sentence) in the `notes/` folder.
